# **App Name**: DevFolio

## Core Features:

- Hero Section: Visually appealing introduction with name, profession, and a brief personal statement.
- Skills Showcase: Interactive display of technical skills (e.g., React, Next.js, JavaScript, HTML, CSS).
- Projects Gallery: A dynamic gallery showcasing past projects with descriptions and links to live demos or GitHub repositories.
- Testimonials: Display positive feedback and recommendations from past clients or collaborators.
- Contact Form: Functional contact form with validation and submission handling.
- AI Style Advisor: Users upload code snippets and this tool offers suggestions on formatting to match a particular design system.

## Style Guidelines:

- Primary color: Vibrant purple (#9F5DE2) to reflect creativity and modernity.
- Background color: Light gray (#F0F0F0) for a clean, professional backdrop.
- Accent color: Electric blue (#5BE1DE) to highlight important elements and calls to action.
- Headline font: 'Poppins' (sans-serif) for a contemporary, precise feel.
- Body font: 'PT Sans' (sans-serif) for readability and a touch of warmth.
- Use modern, minimalist icons for skills, projects, and contact information.
- Subtle animations and transitions for a dynamic and engaging user experience.
